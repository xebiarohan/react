1. Caching in `Next.js`
   - when we build the project, it caches all the static pages.
   - When we start the application it serves those static pages as it is.
   - The downside of this is it never refetches the data on those pages like displaying some list.

2. Revalidating paths
    - So example we added some data in the database, and we are publishing all the added items on a page.
    - We have to `revalidate` that page and all its nested page, so that it can get the latest data
    - By default that page would use cache and not show the latest changes
    - To change that behavior we have to do revalidation when ever we post some data.
    - It takes 2 parameters
      - first parameter is the path of the page that we want to revalidate
      - Second parameter can have 2 values : page (default value) or layout
        - page means revalidate only the current page only
        - layout means revalidate current page and all its nested pages 

```save-data.js
export async function shareMeal(prevValue, formData) {
  const meal = {
    title: formData.get("title"),
    summary: formData.get("summary"),
    creator_email: formData.get("email"),
  };

  if(!meal.title || meal.title.trim() === '') {
    return {
        message: 'Invalid input'
    }
  }

  await saveMeal(meal);
  revalidatePath("/meals", 'layout');
  redirect("/meals");
}

```

3. Metadata
   - we can export metadata from each page like `page.js` or `layout.js`
   - Metadata as name suggests defines the details of the page like title, creator, modify date, etc.
   - We can define the common metadata in the main `layout.js`, it will get applied to all the files.

4. Dynamic metadata
   - we have to create an async function named `generateMetadata`.
   - This function will get called by the `Next.js`.
   - This method gets the `params` that we can use to get the information to pass in the metadata object.
   - Function needs to return an object that will be the final metadata of the class.
   - We can add logic like if the data is not present then programmatically we can call `not-found.js` page.


```dynamic-page.js
export async function generateMetadata({params}) {
  const meal = getMeal(params.slug);

  if (!meal) {
    notFound();
  }

  return {
    title: meal.title
  }
}
```
